# DevOps Deployment Capstone Project

## Project Overview
This project demonstrates a complete CI/CD pipeline setup using:

- React Application
- GitHub
- Jenkins
- Docker
- Nginx

The application is containerized using Docker and deployed through Jenkins automation.
